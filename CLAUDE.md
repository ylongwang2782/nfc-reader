# NFC Card Reader Project

This project provides tools for reading NFC cards via pyscard library.

## Skills

- `/nfc` - NFC card reading operations (UID, Type 4, OneKey Lite)

## Quick Commands

```bash
# Read card UID
./venv_nfc/bin/python scripts/read_uid.py --json uid

# List readers
./venv_nfc/bin/python scripts/read_uid.py --json list
```
